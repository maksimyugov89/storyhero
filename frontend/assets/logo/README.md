# Папка для логотипа StoryHero

## Инструкция

Поместите сюда файл `storyhero_logo.png` с логотипом StoryHero.

### Требования к файлу:
- Формат: PNG
- Размер: минимум 1024x1024 пикселей (рекомендуется 2048x2048)
- Содержимое: Логотип StoryHero с открытой книгой и текстом
- Фон: прозрачный или черный

### После добавления файла:

Выполните команды:
```bash
flutter pub get
flutter pub run flutter_launcher_icons
flutter clean
flutter run
```

Иконки будут автоматически сгенерированы для всех размеров Android.
